### Usage

1. Source the cmake and create a build directory with cmake:

  ```
  cmake -S . -B build
  ```

2. Compile the project:

  ```
  cmake --build build
  ```

3. Run the project:

  ```
  ./build/snaze_release
  ```

There's no way to quit the program use `Ctrl + C` to stop it.
